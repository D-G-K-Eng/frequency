# Frequency

A rhythm-based browser survival game. Enemies pulse on the beat. Stay in sync or die.

## Architecture
- `audio-engine.js` — Web Audio API beat clock + procedural techno loop (Hammerz)
- `game.js` — Canvas renderer, enemy AI, player input (Viv)
- `index.html` — Single-file build

## Dev
Open `index.html` in a browser. No build step required.
